# Cross-chain core library
A complete [monorepos](https://en.wikipedia.org/wiki/Monorepo) Cross-Chain library for multiple Blockchain Network

# Commands
  - Pretty code
  
    ```bash
      npm run pretty:fix
    ```
  - Lint code
  
    ```bash
      npm run lint:fix
    ```
